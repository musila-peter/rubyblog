# musila.dev

Personal portfolio and blog built with [Jekyll](https://jekyllrb.com/) and the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy/) theme.

## Live Site

[musila.dev](https://musila.dev)

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

## Deployment

Deployed automatically to GitHub Pages via GitHub Actions on push to `main`.
